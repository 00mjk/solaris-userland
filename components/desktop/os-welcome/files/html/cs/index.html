Welcome to Oracle Solaris 11 Express

Oracle Solaris is the #1 enterprise operating system for datacenters, developers, and service providers. With proven reliability, performance, security and scalability, it is the best choice to run mission critical applications in an enterprise environment. Take advantage of leading features such as Oracle Solaris ZFS, Crossbow network virtualization and Oracle Solaris Containers to help consolidate your work-loads and lower your administrative costs. Oracle Solaris is the leading operating system for Oracle database, middleware and application deployment providing a fully integrated solution, from application to disk.

Your system will automatically try to connect to a network if you have one available, using both wired and wireless access modes. The network connection is managed through the network auto-magic daemon. If you wish to configure your network manually, check the nwamd(1M) man page for instructions for how to disable this.

We are always looking out for any problems you may experience with this release. If you come across an issue, please help us by reporting a bug. You can also give us feedback on how it runs on your hardware using the Device Detection Utility, and submitting a report!
